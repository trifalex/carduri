# Carduri de Fidelitate

Pagina web personala pentru afisarea propriilor carduri de fidelitate. Aplicatie statica, cu un set fix de carduri.

La click pe un card, codul de bare / QR-ul se deschide pe tot ecranul pentru scanare la casa. Cardurile folosite cel mai recent apar primele, iar primele 4 sunt evidentiate cu o steluta. Exista si un filtru alfabetic.

HTML + CSS + JavaScript vanilla. Fara dependente, fara build step.
